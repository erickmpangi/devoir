installation ROOT

installation Libxrootd-dev

sudo apt install libxrootd-dev
cd ~
git clone https://github.com/xrootd/xrootd.git
cd xrootd
mkdir build
cd build
cmake ..
cmake --build . -- -j$(nproc)
sudo cmake --install .


Avec CONDA :

# Installer Miniconda (si ce n'est pas déjà fait)
wget https://repo.anaconda.com/miniconda/Miniconda3-latest-Linux-x86_64.sh

bash Miniconda3-latest-Linux-x86_64.sh
# Suivre les instructions et recharger le terminal après l'installation.

# Créer un environnement dédié pour ROOT
conda create -n root_env python=3.9
conda activate root_env

# Installer ROOT avec Conda (version stable la plus récente)
conda install -c conda-forge root

Dépendances nécessaires :
